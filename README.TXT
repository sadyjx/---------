本实验将实现如下功能：开机检测ENC28J60，如果检测不成功，则提示报错。在成功检测到ENC28J60之后，初始化uIP，并设置IP地址（192.168.1.16）等，然后监听80端口和1200端口，并尝试连接远程1400端口，80端口用于实现WEB Server功能，1200端口用于实现TCP Server功能，连接1400端口实现TCP Client功能。此时，我们在电脑浏览器输入http://192.168.1.16 ，就可以登录到一个界面，该界面可以控制开发板上两个LED灯的亮灭，还会显示开发板的当前时间以及开发板STM32芯片的温度（每10秒自动刷新一次）。另外，我们通过网络调试软件（做TCP Server时，设置IP地址为：192.168.1.103，端口为1400；做TCP Client时，设置IP地址为：192.168.1.16，端口为1200）同开发板连接，即可实现开发板与网络调试软件之间的数据互发。按KEY0，由开发板的TCP Server端发送数据到电脑的TCP Client端。按KEY2，则由开发板的TCP Client端发送数据到电脑的TCP Server端。LCD显示当前连接状态。

注意：本实验需要用到ALIENTEK ENC28J60网络模块，并请自备网线一根及路由器一个。
